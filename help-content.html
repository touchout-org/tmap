Getting Started

Type an address or place name into the field at the top of the page and press Search (or Enter) to create your first map, centered on that location — this becomes the "anchor POI." Once a map exists, entering another nearby location either adds it to the current map as an additional POI, or, if it's too far away, offers to start a new map centered there instead.

* * note: There are 6 keys on the DotPad. The following documentation uses Perkins-style dot number notation to describe commands entered from the DotPad itself.

Connecting the Dot Pad

Use the Connect Dot Pad button (top of the page) to pair with a physical Dot Pad over Bluetooth — this requires Chrome, since Web Bluetooth isn't available in other browsers.

Steps to connect:

  1. Turn on the Dot Pad and wait for it to complete power-up.
  2. Press the "Connect Dot Pad" button on the web page.
  3. Shift-Tab twice to focus the device selection control.
  4. Press Down arrow to select the Dot Pad device.
  5. Press Enter to connect. If the connection is successful, a grid or map will appear on the display.
  6. Once connected, Disconnect Dot Pad appears at the bottom of the Main Menu.

Drop Pin

Opens via the a hotkey or the Drop Pin button next to the POI list. DotTMAP looks up nearby addresses, tourist attractions, and businesses from OpenStreetMap and lists them alphabetically; the first one fills in the name field automatically. Up/Down step through the rest of the suggestions (they stop at either end, no wraparound), or you can just type your own name. OK always drops the new POI at the cursor's current position, no matter which suggestion is showing; Cancel adds nothing.

Display Preferences

Opens from the Main Menu. Every control here applies immediately — there's no OK/Cancel, just a Done button to close it. Includes Braille Translation, the four braille label checkboxes, Units (Imperial/Metric — affects Scale and how distances are announced), Scale, and Pan Amount.

Map Customization

Opens from the Main Menu (formerly "Edit map..."). Shows or hides individual POIs and streets, lists everything currently hidden in one place, and includes the Map Complexity control — every action here takes effect immediately, with no Save/Cancel step.

My Archives

Opens from the Main Menu. Saved Maps lets you explicitly save the current map under a name with optional notes, then reload, edit, or delete it later — click the Name or Date/Time column header to sort. Recent Maps automatically keeps the last 10 maps you've visited, current map always on top; Clear History empties this list without touching anything you've saved.

The dialog itself opens whether you're signed in or not, but Saved Maps and Recent Maps both require signing in (Login, Main Menu) to store or show anything beyond the current map — sign in first if either section shows a prompt instead of your maps.

Login / Logout

Opens a Google sign-in popup (Login) or signs you out (Logout, shown in its place once signed in) — both live at the same spot in the Main Menu. Signing in only unlocks the Firestore-backed parts of My Archives (Saved Maps and Recent Maps); every other feature works exactly the same either way.
