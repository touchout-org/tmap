DotPad Cursor Performance Experiment

Fixed 60x40 tactile display, no pan or zoom. Move the cursor with the arrow keys or the Dot Pad's cursor dots (3/2/5/6 = left/up/down/right) and use the controls below to test different strategies for keeping the display responsive under rapid cursoring, plus an experimental cursor-acceleration technique that moves further per press once several fast same-direction presses are detected.

This App works best in Chrome. Please switch to Chrome so you can connect to the Dot Pad.

Cursor acceleration (experimental)

After this many fast same-direction presses in a row, the cursor starts moving by the acceleration factor per press instead of 1, and the reference grid hides so only the cursor's own movement is felt. A different direction, or a pause past the timeout, cancels acceleration and restores the grid. Edit a field and press Enter (in any of the four) to apply.
